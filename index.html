Before Start

핵심만 먼저 확인하세요

  • 모드는 개별 검색 설치가 아니라 프로필 코드 Import 방식입니다.
  • Steam 일반 실행이 아니라 r2modman의 Start modded로 실행합니다.
  • 문제가 생기면 Mods 탭과 오류 화면을 캡처해서 넬에게 문의하세요.

Install Flow

설치 흐름

  1. r2modman 설치
  2. Valheim 선택
  3. Import / Update → From code 선택
  4. 넬이 공유한 프로필 코드 붙여넣기
  5. Mods 탭에서 Enabled 상태 확인
  6. Start modded로 실행

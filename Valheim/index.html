전용 프로필 사용

개인 모드 프로필이 아닌 서버용 프로필로 실행해야 접속 오류를 줄일 수 있습니다.

최신 코드 재적용

모드 구성이 바뀌면 기존 프로필을 고치지 말고 최신 코드를 다시 Import해 주세요.

크로스플레이 비활성화

모드 서버 접속 시에는 Valheim 실행 옵션이나 서버 설정에서 Crossplay를 끄는 것을 권장합니다.

캐릭터 백업 권장

기존 캐릭터 사용은 가능하지만 첫 접속 전 백업을 권장합니다.

사망 규칙

장착 장비와 퀵슬롯은 유지되고, 나머지는 묘비에 남습니다. 스킬 레벨 손실은 비활성화되어 있습니다.

레이드 규칙

온라인 인원 3명 이상일 때 레이드가 발생하며, 발생 빈도는 15% 기준입니다. 초원 거점은 레이드 제외 영역으로 운영합니다.

개인 설정

UI 위치와 단축키는 개인 취향으로 조정해도 되지만 서버 싱크 설정은 서버 기준으로 적용됩니다.

Troubleshooting

문제 해결

문제가 생기면 아래 순서로 먼저 확인해 주시고, 해결이 되지 않는다면 초로미에게 문의해주세요.

접속이 안 될 때

  • 서버용 r2modman 프로필로 실행했는지 확인
  • Start vanilla가 아닌 Start modded로 실행했는지 확인
  • 모드 구성이 바뀐 뒤에는 최신 코드로 재Import

버전 불일치가 뜰 때

  • 기존 프로필에서 개별 업데이트하지 않기
  • 프로필을 새로 Import한 뒤 접속
  • 계속 실패하면 오류 메시지 화면 공유

인벤토리/퀵슬롯이 이상할 때

  • AzuExtendedPlayerInventory가 로드됐는지 확인
  • 접속 직후 장비 슬롯과 퀵슬롯 표시 확인
  • 문제 발생 시 캐릭터 백업본 유지

창고 정리가 꼬일 때

  • 상자에 분류용 아이템 1개를 남겨두기
  • 핫바/퀵슬롯 아이템이 자동 보관되는지 확인
  • 빈 상자가 잡템으로 채워지면 설정 공유

포탈 이동이 안 될 때

  • 광석/주괴 이동 가능 여부 확인
  • 포탈 이동 후 무한 로딩이 생기면 재접속
  • 수레/동물 이동은 주변 상태를 정리한 뒤 재시도

로그가 필요할 때

  • BepInEx/LogOutput.log 파일 확인
  • 오류 화면 스크린샷과 함께 공유
  • 재현 순서를 같이 적어주면 빠르게 확인 가능
